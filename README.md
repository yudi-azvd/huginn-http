# Huginn: Servidor HTTP

Servidor HTTP para [Odin](https://odin-lang.org/).

## Executando os exemplos

Em um terminal, inicie o servidor:

```sh
odin run examples/basic
```

Em outra janela de terminal, execute requisições HTTP para o endereço http://localhost:8080. 

Com Python e o pacote `requests`, é possível executar o script que faz requisições simples:

```sh
python client.py
```
